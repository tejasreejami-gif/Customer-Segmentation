# Customer Segmentation Project

## 📌 Overview
Customer Segmentation is a data analytics project that groups customers based on demographic and spending behavior. The goal is to help businesses understand their customers better and develop targeted marketing strategies.

## 🎯 Objectives
- Analyze customer demographics and purchasing behavior
- Identify different customer segments.
- Understand spending patterns.
- Improve customer targeting and retention strategies.
- Visualize customer insights using Power BI.

## 🛠️ Tools & Technologies
- Microsoft Excel
- Power BI
- Python (Optional)
- Data Analytics

## 📊 Dataset Information
The dataset contains **100 customer records** with the following attributes:

- Customer ID
- Age
- Gender
- Annual Income
- Spending Score

## 📈 Dashboard Features
### KPI Cards
- Total Customers
- Average Income
- Average Spending Score

### Visualizations
- Customer Distribution by Gender
- Income Analysis
- Spending Score Analysis
- Age Group Distribution
- Customer Segmentation Overview

### Filters
- Gender
- Age Group
- Income Range

## 🚀 How to Run

1. Download the dataset.
2. Open Power BI Desktop.
3. Import the Excel dataset.
4. Create visualizations and KPI cards.
5. Add slicers and filters.
6. Save the dashboard file (.pbix).

## 📋 Key Insights

- High-income customers have different spending patterns.
- Young customers tend to have higher spending scores.
- Customer segmentation helps businesses target specific customer groups.
- Data visualization provides actionable business insights.
